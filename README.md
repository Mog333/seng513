SENG 513 Project - SnapGram
=======

Getting set up
---
* Make sure you have prerequisites:
  * Node
  * MySQL
* Install Bower globally
  * `npm install -g bower`
* Install local NPM dependencies
  * `npm install`
* Install Bower dependencies
  * `bower install`
